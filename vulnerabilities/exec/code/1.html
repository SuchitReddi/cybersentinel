Command Injection - Demonstration

Here, we made a web application to ping an IP address or domain name, calling the CLI to a `ping` command.

The expected input is an IP address or a domain name. But what about those creative and restless minds out there? They attempt to do something more!
They try to tag additional commands at the end of the IP address. If you aren't responsible to sanitize your input, you may end up giving more control than you want to.

This output tells the malicious user that your website is vulnerable to Command Injection! Now let us see what the vulnerablity is by observing the source code.
