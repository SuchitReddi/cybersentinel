Command Injection - Patching

The bare minimum you want to do will be blacklisting command seperators which enable this vulnerablity.

The best way to patch this vulnerability is by specifying the user input. If you want to take IP address as user input, specify it's exact format.

By specifying the exact format of an IP address and rejecting anything else in the input, there are near-zero chances for someone to gain unwanted access now.
