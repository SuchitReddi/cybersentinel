# CyberSentinel
Demonstration of a collection of various threats and attacks.
https://suchitreddi.github.io/cybersentinel/cybersentinel.html
For tips to setup the application, go to https://suchitreddi.github.io/cybersentinel/docs/setupw.html or /cybersentinel/docs/setupw.html.
After setting up and downloading the repository, rename the folder to "cybersentinel" and place it in htdocs.

## Dockerized CyberSentinel
Dockerized applications can do the downloading and most of the setup part by themselves and are supposedly more secure than using without any virtual box or anything.
Here is the CyberSentinel repository on DockerHub.
Download Docker for your device and follow the instruction in the repository.
